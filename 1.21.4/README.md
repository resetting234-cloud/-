# Сборка для Minecraft 1.21.4 (Fabric)

Полностью совместима с современным стэком Sodium 0.6+:

- **Sodium / Lithium / FerriteCore** — базовая FPS-троица
- **ScalableLux** — многопоточный движок света (заменяет Starlight на 1.21+)
- **Noisium** — оптимизация генерации чанков
- **Exordium** — кэш HUD-кадров
- **C2ME** — параллельная подгрузка чанков
- **ImmediatelyFast** — быстрый рендер GUI/HUD
- **EntityCulling** — отсечение невидимых сущностей
- **ThreadTweak / BadOptimizations / FastChest / FastQuit / Fast IP Ping** —
  мелкие но полезные тюнинги

PvP-моды: **Raised**, **Zoomify**, **MouseTweaks**, **ChatHeads**,
**Rescale (Aspect Ratio)**, **SwingAnimation**, **Health Indicator**.

> Запусти `install.sh` или `install.bat` из корня репо — все моды
> автоматически скачаются в `1.21.4/mods/` и ресурспаки в
> `1.21.4/resourcepacks/`.
