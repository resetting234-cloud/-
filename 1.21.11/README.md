# Сборка для Minecraft 1.21.11 (Fabric)

Самая свежая стабильная ветка 1.21.x. Стэк такой же как у 1.21.5/1.21.8:

- **Sodium 0.6+ / Lithium / FerriteCore** — основа FPS
- **ScalableLux** — light engine, заменяет Starlight
- **Exordium / EntityCulling / ImmediatelyFast / Dynamic FPS** — рендер
- **ThreadTweak / BadOptimizations / FastChest / C2ME / Fast IP Ping** —
  тюнинги

PvP: **Raised**, **Zoomify**, **MouseTweaks**, **ChatHeads**,
**Rescale (Aspect Ratio)**, **SwingThroughGrass**, **Health Indicator**.

> На 1.21.11 некоторые моды ещё не успели зарелизить под последнюю версию —
> установщик честно скипнет их с сообщением «no compatible version».
> Просто проверяй периодически: моды постепенно обновятся.

Запусти `install.sh` / `install.bat` из корня — всё что доступно
скачается в `1.21.11/mods/` + `1.21.11/resourcepacks/`.
